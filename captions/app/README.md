# Ai_Caption_Generator

Author: <Vrushali Deshmukh>
College Project


Ai_Caption_Generator is an Image Captioning System that generates captions for images using Deep Learning techniques.

The system uses:
- CNN (ResNet50) for image feature extraction  
- LSTM for generating meaningful captions  

I trained the model on a custom dataset and tested it on new images to evaluate its performance.

This project demonstrates the combination of Computer Vision and Natural Language Processing to understand image content and generate text descriptions.

## Technologies Used
- Python  
- TensorFlow / Keras  
- ResNet50  
- LSTM  

## Objective
To automatically generate human-like captions for input images.

## Output
The system takes an image as input and produces a descriptive caption as output.
